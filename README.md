# enjmin-vr

![Capture en jeu](./enjmin-vr.png)

## Présentation
L’application est un jeu dans lequel le joueur incarne un personnage qui se déplace d’île en île. 

Sur chaque île, le joueur devra résoudre un puzzle afin de faire apparaître une barque qui emmènera le joueur sur l’île suivante.
